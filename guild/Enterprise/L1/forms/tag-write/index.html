🏷️ Write a tag value

Submits a GitHub Issue pre-filled with the tag write request. The sync-db workflow merges the value into controls/db/tags.json and commits to main.

See controls/scada/gateway/providers/registry.json for the owning provider of each namespace.
Values must be JSON-encoded: "hello", 42, true, null, {"key":"val"}.
Dot-separated namespace path, e.g. room.peerCount, errors.byLevel.ERROR.count
JSON-encoded scalar or object
Module / provider that owns this tag
