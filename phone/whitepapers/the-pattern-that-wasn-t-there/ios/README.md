# The Pattern That Wasn't There · iOS

One-screen SwiftUI app. Part of the ACG whitepaper library.

```bash
brew install xcodegen
xcodegen generate
open ThePatternThatWasnTThere.xcodeproj
```

Uses Xcode 16 (iOS 17 deployment target). Bundle id
`com.aicraftspeopleguild.acg.papers.thepatternthatwasntthere`. Ships to the App Store as an independent app —
every paper gets its own listing.
