# Toastmasters Scrapers Guild · iOS

One-screen SwiftUI app. Part of the ACG whitepaper library.

```bash
brew install xcodegen
xcodegen generate
open ToastmastersScrapersGuild.xcodeproj
```

Uses Xcode 16 (iOS 17 deployment target). Bundle id
`com.aicraftspeopleguild.acg.papers.toastmastersscrapersguild`. Ships to the App Store as an independent app —
every paper gets its own listing.
