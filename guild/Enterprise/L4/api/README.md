# /guild/Enterprise/L4/api

<div align="center"><img src="../../L2/hmi/web/assets/svg/readme-api.svg" alt="Live API endpoints · status chips · CORS open · Pages-served static JSON" width="1040"/></div>

> 📡 **The public API surface.** Every JSON file here is regenerated by the build pipeline and served directly by GitHub Pages. No backend, no auth, CORS open — `curl` works. Browser apps and the fork's own SCADA layer both read from here.
>
> Breadcrumb: [`/`](../../../../) · [`/guild/Enterprise/`](../../) · [`/guild/Enterprise/L4/`](../) · `api/`

## Endpoints

| path | what it is | rebuilt by |
|---|---|---|
| [`health.json`](health.json) | `{paperCount, memberCount, lastUpdated, apiVersion}` — one-shot health check | `scripts/build-api.py` |
| [`papers.json`](papers.json) | every white paper — slug, title, author, date, summary | `scripts/build-api.py` |
| [`members.json`](members.json) | founding members — name, profile slug, joined paper counts | `scripts/build-api.py` |
| [`path.json`](path.json) | URL graph — every deep-link resolved into a nav tree | `scripts/build-api.py` |
| [`state.json`](state.json) | `state.db` snapshot — `summary`, `faults_active`, `generated_at` | `scripts/build-state.py` |
| [`spot-patrol.json`](spot-patrol.json) | last SPOT sweep — `worst`, `summary`, per-beat results | [`build-spot-patrol.py`](../svg/build-spot-patrol.py) |

## Live URLs (fork-served)

```bash
BASE=https://teslasolar.github.io/aicraftspeopleguild.github.io/guild/Enterprise/L4/api
curl $BASE/health.json       | jq .
curl $BASE/state.json        | jq '.faults_active'
curl $BASE/spot-patrol.json  | jq '{worst, summary}'
curl $BASE/papers.json       | jq '.papers[0]'
```
