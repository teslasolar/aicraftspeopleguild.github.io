# Level 0 — Physical Process

> The irreducible source of Guild value: human authoring, thinking, mob sessions.
> See [ISA-95 L0 spec](../../docs/engineering/architecture/isa-95/level-0-physical.md).

## What lives here

Source-of-truth content authored by humans, not generated by automation:

| Asset           | Current path                                    |
|-----------------|-------------------------------------------------|
| Paper originals | `guild/Enterprise/L4/api/white-papers/originals/*.md`         |
| Member profiles | `guild/Enterprise/L4/members/originals/*.md`    |
| AI Rituals      | `guild/Enterprise/L3/scripts/.../ai-rituals` (ingested) |
| Ritual media    | `guild/Enterprise/L4/api/white-papers/*.jpg`, `*.mp4`, `*.png`|
| Paper figures   | in-paper images committed alongside the .md     |

## Nothing is generated here

If a file under L0 changes, it's because a human wrote it. Generated
content (UDT instances, HTML pages, API JSONs) lives elsewhere:

- UDT instances → L4 ERP
- Rendered HTML / screenshots / dist → L3 MES output
- papers.json / members.json → L4 ERP API surface
